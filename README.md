写了一个冒泡排序，思路：随机输入十个数，利用循环输入输出数组。之后将这十个数两两比较，大的放后面这样比较完所有的数，最大的数就被放在了后面，但还是
要总共进行九次外层循环，将九个较大的移到较后的位置上去。同时需要注意的是，C语言的的赋值和Python，尤其看重内存，所以在前后交换上需要三步，用一个变量
暂时储存。
